Game Development Hunters

Unity 3D:

Unity3D is a 3D game development kit with one of the largest asset store to help you progress. The engine supports 3D model formats such as .obj .fbx .3ds etc. Unity3D can use nearly all formats of sound files and images such as .jpg and .png. You can use three different languages that are tweaked for unity such as JavaScript aka UnityScript, C# and boo. The interface is easy to use and learn with a unique function of playing in the edit window with the play and pause buttons at the top of the editor. Unity has pre-set physics and character controllers free to use for projects with a large asset store where you can gather scripts and objects to help in making your game. Uni3D is the ideal engine for indie developers and big company names as well. Here you can find games made with unity.


Unreal Development Kit:

UDK (Unreal Development Kit) is another 3D game development software that supplies excellent lighting, animations, audio and everything that could make a game look good. UDK also has their own kind of asset store where you can get models, lights and all sorts of game related software. The editor has a base layout like another 3D modelling program called SketchUp, with tools on the left and top of the window. These tools lets you create terrain, levels and object transformers. The engine can support .fbx 3D files and sounds.

Game Maker:

GameMaker is a 2D game creating software that is great for users to start from. The editor is simple to use with a sprite, room, object, sound and code editor. It is possible to make 3D games but you must first buy GameMaker and then hard code the 3D environment which makes this program only ideal for the development of 2D games. Fixel:

Flixel is a basic 2D game development kit which is great for starters who know nothing about game development or programming. You can use a variety of tools such as saves, particle effects, simple collisions, path finding and much more simple to use tools.
